    FileSystem

    FileSystem Class - Use $fs Wraps ArrayBuffers in Buffer helper class

    Example:

    // In samp.js $fs is an instance of FileSystem
$fs = new FileSystem();

// Opens myfile.txt and reads 1000 bytes of it
let fd = $fs.open("myfile.txt");
let buf = $fs.read(fd,0,1000); 
$fs.close(fd);

let str = buf.toString();

    public readFile(filename: String, callback: function(data: Buffer)): boolean source

    Reads an entire file into a buffer

    Params:

    NameTypeAttributeDescription
    filename String

    Filename you want to read
    callback function(data: Buffer)
    • optional

    when a callback is supplied file will be read asyncronously

    Return:

    boolean

    Return a buffer object (ArrayBuffer wrapper) / True/False in async mode

    Example:

    // Read file asyncronously and fire callback when finished
$fs.readFile("myfile.txt", function(buffer){
    print(buffer.toString());
});

// Read file syncronously
print($fs.readFile("myfile.txt").toString());
